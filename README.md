# Intern Backend Project

## Tech Stack
- Node.js
- Express.js
- MySQL
- React

## Features
- User Registration & Login (JWT)
- Password hashing (bcrypt)
- Task CRUD (Create, Read, Update, Delete)
- Protected routes
- Simple UI

## Setup Instructions

### Backend
cd backend  
npm install  
node server.js  

### Frontend
cd frontend  
npm install  
npm start  

## API Endpoints
- POST /api/v1/register
- POST /api/v1/login
- GET /api/v1/tasks
- POST /api/v1/tasks
- PUT /api/v1/tasks/:id
- DELETE /api/v1/tasks/:id
